2.1 通信基础

1.计算机内部的数据传输是什么传输？

计算机内部的数据传输通常采用并行传输方式。并行传输是指数据的多个位同时通过多条传输线路进行传输，这种方式能够显著提高数据传输速度。例如，CPU与内存之间的数据传输就是通过总线进行的，常见的数据总线宽度有8位、16位、32位和64位等。

串行传输与并行传输

  • 串行传输：数据按位依次传输，只需一条传输信道。该方式节省线路和设备，适合远程传输，广泛应用于通信网和计算机网络中。
  • 并行传输：数据在多条信道上同时传输，传输速率高，但需要多条传输线路，成本较高，通常用于短距离传输，如计算机内部的总线传输。

单工、半双工与全双工通信

  • 单工通信：数据只能单向传输。例如无线广播。
  • 半双工通信：数据可以双向传输，但不能同时进行。例如对讲机。
  • 全双工通信：数据可以同时双向传输。例如电话通信。

同步传输与异步传输

  • 同步传输：数据以稳定的比特流传输，字节之间没有间隔，要求收发双方的时钟同步。
  • 异步传输：以字节为单位传输，字节之间时间间隔不固定，每个字节前后有起始位和结束位。

基带传输、频带传输与数字数据传输

  • 基带传输：基带信号直接在信道上传输，不进行频谱搬移。
  • 频带传输：基带信号经过调制后传输，适用于模拟信号。
  • 数字数据传输：利用PCM（脉冲编码调制）信道传输数据信号。

其他传输方式

  • 光纤传输：利用光通信技术，传输速度快，但成本较高。
  • 无线传输：方便灵活，但受传输距离和干扰限制。

2.曼彻斯特编码的叙述正确的是？说明原因

  1. 每个信号的起始边界作为时钟信号有利于信号同步（X）中间的跳变作为时钟信号。
  2. 将时钟信号和数据取值包含在信号中（√）码元的和中间跳变作为时钟信号；不同的跳变方式“上0下1”作为数据信号。
  3. 适合传输模拟数据（X）具体来说应该是二进制数据。

3.使用曼彻斯特编码作为数据通信的主要原因是：

答：实现对通信过程中收发双方的数据同步。（跳变表示bit，双发根据跳变来同步时钟，不需要额外时钟信号）

4.不含同步信息的编码

非零编码NR不包括同步信息（因为各个码元之间没有间隔标志，他的同步信息由时钟线来同步）

5.采样率带宽之间有什么关系？

采样率>=2*带宽(Hz)

别忘了带宽如果以Hz为单位的话，它的定义是带宽的频率，就是那高带宽-低带宽。

6.香农定理的局限性

如果告诉你这个是二进制信号，那在使用香农定理的时候还要计算奈氏准则（限制了码元可以携带bit位数），取两者的最小值。

2.2 传输介质

传输介质可以分为

  • 导向传输介质：电磁波沿固体介质传播。
  • 非导向传输介质：沿自由空间传播。

1.双绞线

绞合的作用？

减少电磁干扰。

如何进一步提高电磁干扰能力？

在双绞线外面加上一层屏蔽层：

  • 屏蔽双绞线（STP）
  • 非屏蔽双绞线（UTP）

双绞线带宽取决于什么？

双绞线的带宽取决于铜线的粗细和传输的距离。

  • 距离太远。
    • 对于模拟信号要使用放大器。
    • 对于数字信号使用中继器进行整形。

2.同轴电缆

同轴电缆组成？

由内导体、绝缘层、外导体屏蔽层和绝缘保护套层构成。

同轴电缆为什么比双绞线传输速率更快？

同轴电缆具有更高的屏蔽性，同时具有更高的抗噪声性。

3.光纤

光纤的组成？

由纤芯和包层构成。

光纤利用光的什么特性？

全反射特性。

光纤的特点？

  • 传输损耗小，中继距离长，远距离传输经济。
  • 抗雷电和电磁干扰性能好。
  • 无串音干扰，保密性好，不易被窃听。
  • 体积小重量轻。

2.2.2 物理层接口的特性

  • 机械特性：接口所用接线器的形状与尺寸、引脚****数量和排列、固定和锁定装置等
  • 电气特性：某接口电缆在各条线上的电压范围、纯属速率和距离限制等。
  • 功能特性：某条线上某一电平的电压意义。
  • 过程特性：也成为规律特性，置不同功能的各种可能时间当出现顺序

举个例子，信号电平用10V到15V表示二进制0是——电气特性；但是11V表示二进制0就是功能特性。

2.3 物理层设备

2.3.1 中继器

中继器功能

放大、整形并转发信号。消除信号经过一长段电缆后产生的失真和衰减。

注意其是信号再生，而不是简单的放大信号！

中继器不可以无限串联

根据10BASE5以太网规范中，遵循5-4-3原则：互相串联的中继器的个数不超过4个，且其串联起来的5段通信介质中只有3段可以挂载计算机。

中继器与放大器的区别？

中继器（Repeater）是一种用于放大和再生****数字信号的设备，主要用于在长距离传输中保持信号的完整性和强度。中继器通过接收信号、放大信号并重新发送信号来克服信号衰减和噪声的影响，但它只适用于数字信号，因为数字信号可以被精确地重建。

对于模拟信号，通常使用放大器（Amplifier****）来放大信号。放大器会增强模拟信号的幅度，但同时也会放大信号中的噪声和失真。与中继器不同，放大器不会对信号进行重建，而是直接增强信号的强度。

总结：

  • 中继器：用于放大和再生数字信号。
  • 放大器：用于放大模拟信号，但不会重建信号。

2.3.2 集线器

什么是集线器

集线器本质上是一个多端口的中继器。

总结一下：中继器通常只有2个端口，而集线器通常由四个或更多。

集线器与中继器所在的层？

都是物理层。

集线器与冲突域和广播域

集线器连接的工作站集合同属于一个冲突域和一个广播域。

集线器可以连接两个怎样的局域网？

集线器可以连接两个不同介质的局域网。如一个是双绞线，另一个是光纤。

但是不能连接两个速率不同，或者是协议不同的局域网。

  • 速率不同，数据包可能会因为速率不匹配而发生冲突或丢失。
  • 不同协议可能使用不同的信号编码方式（如曼彻斯特编码、NRZ编码等）。即使集线器将信号传递到目标设备，目标设备也可能无法正确解码这些信号。

五台机器连接10Mb/s的集线器上，每台计算机分的的平均带宽至多为多少？

由于集线器是通过广播的形式发送信号的，所以要平分，也就是10Mb/s/5 = 2Mb/s。
